# 👁️ RbxAntiAFK - Premium & Open Source AFK Solution

**RbxAntiAFK** is a lightweight, high-performance utility designed to prevent you from being kicked for inactivity in Roblox. By simulating natural user behavior, it allows you to stay on servers 24/7. Since it operates as an external macro, it does not inject into the game, keeping your account safe.

![Main UI](https://ibb.co/N6q1pG6h)

## ✨ Key Features
- 🕒 **Custom Intervals**: Set your own action frequency in minutes.
- 🔇 **Silent Mode**: Runs in the background without interrupting your workflow.
- 💬 **Discord Integration**: Receive real-time status reports via Webhooks.
- 📸 **Remote Screenshots**: Get visual proof of your game status sent directly to your Discord.
- 🎮 **Rich Presence (RPC)**: Showcase your "Grinding" status to your Discord friends.
- 🔓 **100% Open Source**: Fully transparent code for maximum trust.

## 🛡️ Security & Antivirus Notes
Some antivirus engines may flag this application as a **RAT** or **Trojan**.
**These are False Positive detections** caused by the nature of macro automation:
- **Mouse Movement**: Using `mouse_event` to simulate activity.
- **Screenshots**: Capturing the game window for remote monitoring.
- **Webhooks**: Sending data to Discord servers.

The project is fully open-source; you are encouraged to review the code yourself to verify its safety.

## 🚀 Getting Started
1. Download the latest build from the [Releases]([https://github.com/your-username/RbxAntiAfk/releases) section.
2. Launch `RbxAntiAfk.exe`.
3. Click the **START** button.
4. Enter your desired interval (in minutes).
5. (Optional) Paste your Discord Webhook URL to enable remote logging.
